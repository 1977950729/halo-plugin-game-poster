# 概述
本项目旨在编写插件实现个人游戏海报墙功能，为 Halo 博客提供单独的页面用来展示个人玩过的游戏和页面。列表中提供 steam、switch 等游戏目录。

# 功能
## 1. 显示 steam 相关内容
steam 作为 pc 游戏平台。

### 1.1 显示已拥有的游戏
页面中需要展示博主拥有的游戏，并可以通过一些配置选择过滤某些游戏

### 1.2 显示当前状态
需要提供显示当前游戏状态的接口，用于展示博主的 `steam` 在线及游戏状态。

### 1.3 显示游戏详情
需要在游戏详情页面上显示博主的游戏状态，包括
- 最近游玩时间
- 成就
